## OTA - Over The Air:
▪ It's a simple way to burn your hex file into a microcontroller by a website (instead of offline software) using a bootloader to receive hex code from the server. \
▪ Using: STM32F10C8T6 (ARM Cortex-M3) - ESP8266.
